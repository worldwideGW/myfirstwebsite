엑셀 스터디

목적
* 엑셀 업무 시, 작업 효율을 높인다
* 데이터를 빠르게 기입하고, 이를 시각화하기 위해 필요한 스킬을 키운다
* 엑셀 작업 속도를 높여주는 숏컷을 빠르게 습득한다

가장 자주 사용하는 함수 모음
1. COUNT : 선택한 셀에서 숫자만 감지하여 카운트해준다
2. SUM :
3. IF : =IF(조건식, 참일때 출력, 거짓일때 출력)
4. AVERAGE : =AVERAGE(range)
4. COUNTIF : =COUNTIF(range, 조건식)
5. SUMIF : =SUMIF(조건이 적용되는 range, 조건식, SUM을 적용할 range)
